# CoinGame

### コインプッシャーゲームのサンプル
___

### 遊び方
___
コインを左右どっちかから投入して
プッシャーとコインを駆使してコインを下に落とす。

自分のコインの枚数がゼロになったらゲームオーバー。

### 主なコードの記述
___
> 初期コインの生成
>> `Randomcoin`に記述

> コインの投入
>> `Shot`に記述

> メダル全体のManager
>> `MedalManager`に記述
